<!-- v1 已占用 msg_type（节选）— 执行器不得将 PROTOCOL_ERROR 设为 0x10 -->

| 值（hex） | 名称 |
|-----------|------|
| `0x01`–`0x04` | `SESSION_*`（session-create-join.md） |
| `0x05` | **本计划分配：`PROTOCOL_ERROR`** |
| `0x10`–`0x12` | `STREAM_OPEN` / `STREAM_DATA` / `STREAM_CLOSE`（streams-lifecycle.md） |

**PROTOCOL_ERROR 消息体（控制消息，无路由前缀）：**

| 字段 | 长度 | 类型 |
|------|------|------|
| `err_code` | 2 | uint16 BE |
| `reason_len` | 2 | uint16 BE |
| `reason` | `reason_len` | UTF-8；`reason_len=0` 时省略 |
</interfaces>
